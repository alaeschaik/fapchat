# fapchat
A simple chat tool.
## Gradle

### Linux / Mac

If you run on Linux or Mac, follow these steps:

    cd fapchat
    
To run the project (you might have to grant execution rights to gradlew):
    
    ./gradlew run

### Windows

If you run on Windows, follow these steps:

    cd fapchat

To run the project:
    
    gradlew run
